# 答案之书 - 快速启动指南

## 🚀 快速体验

### 方法一：使用演示脚本（推荐）
```bash
cd wisdom-book
node demo.js
```
然后在浏览器中访问：http://localhost:3004

### 方法二：使用完整开发环境

1. **安装依赖**：
```bash
npm run install-all
```

2. **启动开发服务器**：
```bash
npm run dev
```

3. **访问应用**：
- 前端：http://localhost:3000
- 后端API：http://localhost:5000

## 📱 功能特点

✅ **基于时间的动态内容** - 根据访问时间生成不同的哲理建议
✅ **每次点击都有新内容** - 不会重复相同的建议
✅ **积极向上的内容** - 所有建议都经过精心筛选，健康积极
✅ **简洁美观的界面** - 现代化的设计风格
✅ **完全响应式** - 支持手机、平板、桌面设备
✅ **无需登录** - 直接访问即可使用
