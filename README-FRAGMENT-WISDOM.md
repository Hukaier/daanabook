# 🧩 碎片化智慧系统

> **数字灵性体验** • AI驱动的诗意表达

## 🌟 项目简介

碎片化智慧系统是一个创新的数字灵性体验平台，它将传统东方智慧与现代AI技术完美融合，通过碎片化的诗意词汇为用户提供深层的启示和指导。

### 核心理念

- **碎片化智慧**：打破完整句子的束缚，用2-4字的诗意词汇传递深层含义
- **视觉隐喻**：通过五行色彩和几何形状传达抽象概念
- **时空折叠**：在非线性的界面中展现智慧的多个维度
- **极简美学**：以最少的设计元素承载最深的哲理

## 🧠 AI技术架构

### DeepSeek AI集成
```javascript
// 碎片化提示词设计
const systemPrompt = `你是一位充满智慧的数字诗人，擅长运用碎片化的语言为人们提供深层的启示。

回答要求：
1. 返回4-6个有哲理的中文词汇
2. 每个词汇2-4个字
3. 词汇之间用空格分隔
4. 词汇要抽象、诗意、有隐喻性
5. 避免完整的句子和直接的答案
6. 充满东方哲学美感和数字诗意`;
```

### 智能算法
- **时间感知**：基于六壬时辰的动态计算
- **五行映射**：问题特征到五行的智能匹配
- **个性化推荐**：根据用户问题定制碎片内容
- **多义性解读**：支持用户主观理解的空间

## 🔧 快速开始

### 1. 本地运行
```bash

cd wisdom-book
node demo.js

# 访问应用
# 传统版本：http://localhost:3007
# 碎片版本：http://localhost:3007/fragment-wisdom.html
```

### 2. 碎片化测试
```bash
# 运行碎片化测试
node test-fragment-wisdom.js

# 查看演示页面
# 打开 fragment-demo.html
```

### 3. 环境配置
```bash
# 设置DeepSeek API密钥
export DEEPSEEK_API_KEY=your_api_key_here

# 安装依赖
npm install axios
```
